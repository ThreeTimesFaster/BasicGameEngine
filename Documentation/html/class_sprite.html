Parameters

The color alignment to which the sprite is drawn according to. Setting it to Color::White outputs the sprites base color, while setting it to Color::Black outputs no color.

The type of blending mode that is rendered for the sprite. Can apply three types of blending; Alpha, Add and Multiply. Example input: BlendingMode::Alpha
